# Exercises

Repository dedicated to small programming exercises I designed to practice core concepts.

Most exercises focus on **JavaScript**, with occasional use of **HTML** and **CSS** when necessary.

## Purpose

- Practice logic and problem-solving
- Improve JavaScript fundamentals
- Experiment with small ideas before building larger projects

## Scope

This repository contains only small, isolated exercises.  
Larger projects (e.g., projects like *Flappy Bird with JavaScript*) will have their own dedicated repositories.

## Structure

Each exercise lives in its own folder and may contain:
- `script.js`
- `index.html`
- `style.css` (if needed)
